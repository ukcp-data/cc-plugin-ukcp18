# Installing the UKCP18 plugin and its dependencies

## Use the script

In order to install there are various components that you need in place.

Please use the install script to do this:

```
$ wget https://raw.githubusercontent.com/ukcp-data/cc-plugin-ukcp18/master/scripts/get-ukcp18-checker.sh 
$ ./get-ukcp18-checker.sh --install 
```

## Updating the plugin (and vocabs)

Use the code (from GitHub) and CVs with `--update`:

```
$ ./get-ukcp18-checker.sh --update
```

## Running the checker

Run the checker with:

```
$ source setup_env.sh
$ compliance-checker --test ukcp18-file-info --test ukcp18-file-structure --test ukcp18-global-attrs cc_plugin_ukcp18/tests/data/ukcp18/test_cdl_global_atts.nc 
```

## Running the unit tests

To run the `checklib` unit tests, do:

```
$ source setup_env.sh
$ cd ../check-maker/
$ pip install pytest
$ py.test
```
